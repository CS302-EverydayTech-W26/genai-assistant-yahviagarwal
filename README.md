To use the Gemini AI Assitant on my Webpage: 
1. Enter the URL http://127.0.0.1:5000
2. On the form Enter your prompt/question.
3. Click the Ask AI button.
4. See response on screen.
5. If you want to ask another question, clear the previously asked question and type in the new question
   and click the ask AI button.
